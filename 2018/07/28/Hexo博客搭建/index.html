目前的操作

  1. clone hexo分支 （新电脑）
  2. npm install（新电脑）
  3. 写博客 hexo n “博客名”
  4. hexo g 生成博客
  5. hexo s启动网站
  6. hexo d发布到master
  7. 提交hexo内容到hexo分支
